# QBR Slide Deck Template

## Overview

Standard template for Quarterly Business Review presentations.

## Slide Structure

### Slide 1: Title
```
[Customer Logo]

Quarterly Business Review
[Quarter] [Year]

[Account Manager Name]
[Customer Success Manager Name]
```

### Slide 2: Executive Summary
```
## Key Accomplishments This Quarter

- [Accomplishment 1 with metric]
- [Accomplishment 2 with metric]
- [Accomplishment 3 with metric]

## Business Impact
[1-2 sentence summary]

## Next Quarter Focus
[Primary goal]
```

### Slide 3: Usage Overview
```
## Product Usage

[Usage Trend Chart]

| Metric | This Quarter | vs Last Quarter |
|--------|--------------|-----------------|
| Active Users | [number] | [+/- %] |
| Total Events | [number] | [+/- %] |
| Workflows Run | [number] | [+/- %] |
```

### Slide 4: Key Use Cases
```
## Use Cases in Production

### [Use Case 1]
- Business Outcome: [result]
- Quote: "[customer quote]" - [Role]

### [Use Case 2]
- Business Outcome: [result]
- Adoption: [% of users]
```

### Slide 5: Forward-Looking
```
## Discussed, Not Yet Implemented

- [Opportunity 1]
- [Opportunity 2]

## Recommendations for Next Quarter

1. [Recommendation 1]
2. [Recommendation 2]
```

### Slide 6: Credit Pacing
```
## Usage vs Contract

[Chart showing usage vs limits]

## Expansion Opportunities

- [Opportunity 1]: [estimated value]
- [Opportunity 2]: [estimated value]
```

### Slide 7: Appendix
```
## Data Sources

- Product Usage: Snowflake (as of [date])
- CRM: Salesforce (as of [date])
- Conversations: Gong (last 90 days)
- Support: Zendesk (last 90 days)
```
